# twitch_chat

Package to connect and use the Twitch Chat by Websocket and IRC.

## Features

- [x] Connect to chat
- [x] Connect anonymously
- [x] Get badges
- [x] Get emotes
- [x] Get BTTV, FFZ & 7TV emotes
- [x] Ban, Timeout, Delete message
